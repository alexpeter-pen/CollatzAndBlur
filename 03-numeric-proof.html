3 · The Numeric Proof

What counts as a computer‑assisted certificate.

Home ← Prev Next →

Goal in one sentence

Produce a finite certificate that forces a uniform one‑step decrease of a Lyapunov scale \(V(n)=\log n + \varphi(n\bmod 2^k)\) along all accelerated odd steps. The certificate is a collection of linear inequalities indexed by odd residues modulo \(2^k\).

The inequality

Write the accelerated step for odd \(n\) as \(T(n)=\dfrac{3n+1}{2^{v_2(3n+1)}}\). We want

\(V(T(n)) - V(n) = [\log 3 - v_2(3n+1)\cdot\log 2 + \log(1+1/(3n))] + [φ(r') - φ(r)] ≤ -δ,\)

where \(r\equiv n\pmod{2^k}\) (odd) and \(r' \equiv T(n)\pmod{2^k}\). The tiny term \(\log(1+1/(3n))\) is bounded from above uniformly on each residue class by a small budget \(\rho_k\). Thus it suffices to enforce, for every odd residue \(r\),

\(φ(r') - φ(r) ≤ -δ - [\log 3 - v_2(3r+1)\cdot \log 2] + ρ_k.\)

These are linear constraints in the unknowns \(\{φ(r)\}\) and \(δ\).

How we obtain the certificate

  1. Fix a modulus \(2^k\) (e.g., start small; later we use \(k=13\)). Unknowns: one value \(φ(r)\) per odd residue and a margin \(δ\).
  2. Compute transitions \(r\mapsto r'\) and exponents \(v_2(3r+1)\) for all odd residues.
  3. Choose a conservative bound \(\rho_k\) so that for all \(n\equiv r\pmod{2^k}\), \(\log(1+1/(3n))\le \rho_k\) (checked by interval arithmetic).
  4. Solve the LP: maximize \(δ\) subject to all residue inequalities and a normalization (e.g., average \(φ\)=0).
  5. Export a certificate: the list of \(φ(r)\), the value of \(δ\), the bound \(\rho_k\), and \(k\).
  6. Independent verify: a tiny program re‑derives \(r'\), checks every inequality with interval arithmetic (no rounding trust), and confirms \(δ>0\).

Philosophically: this is a finite, checkable witness that average drift is negative after one accelerated step.

Toy example (k=4, schematic)

For \(k=4\) there are 8 odd residues. One constraint looks like

φ( (3·r+1)/2^{v_2(3r+1)}  mod 16 ) - φ(r)
≤ -δ - [log 3 - v_2(3r+1)·log 2] + ρ₄.

Solving the tiny LP may already give a small positive \(δ\). Larger \(k\) improves uniformity while keeping the system finite.

What “numeric proof” means here

  • Finite: everything reduces to a finite LP and finitely many interval checks.
  • Auditable: a separate, minimal checker verifies the output without relying on a solver’s internals.
  • Stable: the checker uses interval arithmetic to prevent false positives from floating‑point error.

In later pages we’ll connect \(δ\) and the switching budget \(\rho\) (from +/× lens mismatch) and discuss thresholds where drift guarantees convergence.
