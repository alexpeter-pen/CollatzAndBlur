10 · Slow Derivation to a Proof

Two routes that meet in the same place.

Setup

Work with the accelerated odd map \(T(n)=\dfrac{3n+1}{2^{v_2(3n+1)}}\) and the Lyapunov scale \(V(n)=\log n + \varphi(n\bmod 2^k)\). Define the jagged term \(D_0(r)=\log 3 - v_2(3r{+}1)\,\log 2\) on odd residues \(r\in (\mathbb Z/2^k\mathbb Z)^{\times}_{\text{odd}}\), pick a short positive blur \(\mathcal B\), and write \(\overline D=(\mathcal B D_0)\). The small additive correction \(\varepsilon(n)=\log(1+1/(3n))\) is bounded by \(\rho\) after the same local blur.

Route A — derive the inequalities

  1. Residue graph: compute \(r\mapsto r'\equiv T(n)\pmod{2^k}\) for all odd \(r\); for the unique \(r^*\) with \(3r^*+1\equiv0\pmod{2^k}\), use \(v_2\ge k\).
  2. Blurred drift: enforce for every odd residue \[ \varphi(r')-\varphi(r) \;\le\; -\delta - \overline D(r) + \rho. \]
  3. Normalization: fix the gauge of \(\varphi\) (e.g., average zero) to make the system well-posed.
  4. LP: solve the finite linear program for \(\{\varphi(r)\}\) and margin \(\delta>0\).

Any \(\delta>0\) with a small \(\rho\) yields a uniform one‑step drop of \(V\).

Route B — verify a candidate table

Start from a proposed table of \(\varphi(r)\), a window \(\mathcal B\), and parameters \(k,\rho,\delta\). A tiny checker:

for each odd residue r:
  e  = v2(3*r+1)              # with e >= k when r == r*
  D0 = log(3) - e*log(2)
  D  = (ℬ D0)(r)              # local blur over odd neighbors
  lhs = φ(r') - φ(r)          # r' = T(n) mod 2^k
  rhs = -δ - D + ρ
  assert lhs <= rhs + tol     # interval arithmetic to bound roundoff

Interval arithmetic (directed rounding) ensures the checker is sound even with floating‑point inputs.

From one‑step drift to termination

  • Summing \(V(T(n)) - V(n) \le -\delta\) along odd steps, \(V\) drops at least \(m\delta\) after \(m\) steps.
  • Hence \(\log n\) eventually falls below a finite threshold; from there a short finite computation reaches \(1\).
  • Even steps (halving) are benign; they only help.

Certificate format

  • Parameters: \(k,\; \rho,\; \delta,\; \mathcal B\) (window and weights).
  • Table: \(\{\varphi(r)\}_{r\,\text{odd mod }2^k}\).
  • Verification log: pass/fail for each residue inequality, with interval bounds.

Either route delivers the same artifact: a finite, auditable certificate of uniform drift.
