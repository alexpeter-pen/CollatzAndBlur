The issue: jagged residue drops

The one‚Äëstep change \[ \Delta(n) \,=\, \log\frac{T(n)}{n} \,=\, \underbrace{\log 3 - v_2(3n{+}1)\,\log 2}_{\text{residue‚Äëdriven}} \;+\; \underbrace{\log\bigl(1+\tfrac{1}{3n}\bigr)}_{\text{tiny additive}}. \] The first term is piecewise constant on residues \(n\bmod 2^k\) but spiky; the second is tiny but depends on \(n\) itself. Without smoothing, a single ‚Äúbad‚Äù residue can cancel average drift.

The blur operator

Pick a short, positive window \(W=\{-m,\ldots,m\}\) and weights \(w_h\ge 0\) with \(\sum_h w_h=1\). For any function on odd residues \(f:\ (\mathbb Z/2^k\mathbb Z)^{\times}_{\text{odd}}\to\mathbb R\), define

\( (\mathcal B f)(r) \,=\, \sum\limits_{h=-m}^m w_h\, f(r+2h) \quad (\bmod\ 2^k). \)

  • Locality: only nearby last bits are averaged (odd residues differ by steps of 2).
  • Positivity: \(w_h\ge 0\) prevents spurious cancellation; the certificate stays conservative.
  • Finite: all data live modulo \(2^k\); no unbounded tails appear.

What we blur and why

  • Jagged part: set \(D_0(r)=\log 3- v_2(3r{+}1)\log 2\) and use \(\overline D(r)=(\mathcal B D_0)(r)\).
  • Tiny additive: the correction \(\varepsilon(n)=\log(1+1/(3n))\) admits a uniform bound on each residue class after the same local blur; we denote it by \(\rho\).

\(\rho\) is the principled ‚Äúswitch budget‚Äù for moving between the additive and multiplicative lenses.

If some \(\delta>0\) exists with \(\rho+\delta<\log(4/3)\) (or a similar explicit threshold), then \(V\) drops uniformly on odd steps.

Choosing the window

  • Boxcar (uniform) weights: simplest, positive, and respects locality; good default.
  • Radius \(m\): small compared to \(2^k\) so we only mix nearby last bits; larger \(m\) reduces jaggedness but increases \(\rho\).
  • Normalization: fix \(\sum w_h=1\) and a gauge for \(\varphi\) (e.g., average zero) to keep the LP well‚Äëposed.

Verification path (sketch)

  1. Fix \(k\), window \(W\), and weights \(w_h\).
  2. Compute \(\overline D(r)\) and the residue graph \(r\mapsto r'\) (care with the exceptional \(r^*\)).
  3. Bound \(\varepsilon(n)\) by \(\rho\) on each class using interval arithmetic.
  4. Solve the finite LP for \(\{\varphi(r)\}\) and \(\delta\); export and independently verify the inequalities.
