11 · The Special Class r*

Why one residue is exceptional — and how we handle it safely.

Home ← Prev Next →

Existence & uniqueness

Fix \(k\). Because \(3\) is invertible modulo \(2^k\), there is a unique odd residue \(r^*\) with \[ 3r^* + 1 \equiv 0 \pmod{2^k} \quad\Longleftrightarrow\quad r^* \equiv -3^{-1} \pmod{2^k}. \] This is the class where the 2‑adic valuation can exceed the modulus bound.

Compute \(3^{-1}\bmod 2^k\) by Hensel/Newton lifting: starting from 1 mod 2, iterate \(a\leftarrow a(2-3a)\) mod \(2^t\) doubling \(t\).

Consequence for \(v_2(3n+1)\)

For all \(n\equiv r^*\pmod{2^k}\) we know \[ v_2(3n+1) \ge k, \] with any extra 2‑adic carries depending on higher bits of \(n\). Thus the residue‑driven drop \(\log 3 - v_2(3n{+}1)\,\log 2\) is bounded above by the conservative value \(\log 3 - k\log 2\) on this class.

Certificate constraints at \(r^*\)

With \(V(n)=\log n + \varphi(n\bmod 2^k)\) and the accelerated step \(T(n)=\dfrac{3n+1}{2^{v_2(3n+1)}}\) we must ensure, for every possible outgoing residue \(r'\equiv T(n)\pmod{2^k}\),

φ(r′) − φ(r*) ≤ −δ − [log 3 − k·log 2] + ρ.
  • Safe option A (max‑neighbor): enforce the inequality with \(φ(r′)\) replaced by \(\max_s φ(s)\). Conservative but simple.
  • Safe option B (resolve carries): work at modulus \(2^{k+L}\) to determine the exact set of \(r'\) for \(r^*\); enforce constraints for all those \(r'\), then project back to \(2^k\).

Either option keeps the certificate finite and sound; extra 2‑adic factors only help the RHS.

Takeaways

  • There is exactly one exceptional odd residue \(r^*\) at each \(k\).
  • Use \(\log 3 - k\log 2\) for its bracketed term; treat \(r'\) conservatively (A) or explicitly (B).
  • All other residues have determined \(v_2(3r{+}1)\) and \(r'\) mod \(2^k\).
