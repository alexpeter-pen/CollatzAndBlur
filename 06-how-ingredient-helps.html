Split the step, then budget the switch

Write the accelerated odd step as \(T(n)=\dfrac{3n+1}{2^{v_2(3n+1)}}\). Its one‑step change \(\Delta(n)=\log\tfrac{T(n)}{n}\) splits into a large residue‑driven term \(\log 3-v_2(3n{+}1)\log 2\) and a tiny additive correction \(\log(1+1/(3n))\). A tiny, explicit blur over the last bits lets us bound the correction by \(\rho\) uniformly on residue classes.

We won’t optimize \(\rho\) here—this is a small, didactic preview.

Micro‑demo: first 64 odd residues inside 2^13 arithmetic

This illustrates how a small last‑bits blur stabilizes the jagged residue term—exact thresholds and the \(\varphi\) correction come next.

Where this is going

With a Lyapunov scale \(V(n)=\log n + \varphi(n\bmod 2^k)\), we’ll pick \(\varphi\) by solving a finite LP so that \(V(T(n)) - V(n) \le -\delta\) holds uniformly, using the blur budget \(\rho\). We will later work at \(k=13\) fully; the 64‑residue preview here only demonstrates the stabilizing effect.
